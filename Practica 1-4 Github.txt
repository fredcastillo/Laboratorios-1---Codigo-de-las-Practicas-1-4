Comandos usados practica 3

*Comandos de Administración (Root)*

• sudo su
• useradd -m FredCastillo
• cat /etc/shadow
• passwd FredCastillo
• usermod -aG wheel FredCastillo
• su - FredCastillo
• sudo su - Fredcastillo   
• exit

*Creación y Adición de Usuario y Grupo*

• groupadd Guest
• cat /etc/group
• useradd modest
• usermod -aG Guest modest

Comandos de Eliminación

• userdel -r modest
• cat /etc/shadow
• groupdel Guest
• cat /etc/group

Comandos usados practica 4


Creación y Edición

• mkdir materia
• cd materia
• touch estudiante.txt
• nano estudiante.txt
• cat estudiante.txt
• ls -l

Modificación de Permisos
• chmod 700 estudiante.txt (Control total solo para el usuario)
• chmod 070 estudiante.txt (Control total solo para el grupo)
• chmod -x estudiante 

Manipulación y Eliminación

• cd ..
• mkdir materias2
• cp materia/estudiante.txt materias2/
• rm -r materias2
• rm materia/estudiante.txt
• rmdir materia
